# **Simple Multithreaded Web Server**

A lightweight, custom-built multithreaded web server implemented in Java.
It can handle multiple client connections simultaneously using threads, serve basic HTTP responses, and provide a foundation for understanding low-level server mechanics.

---

## Features

### Multithreaded Request Handling

* Each client request is processed in its own thread
* Supports concurrent connections

### Basic HTTP Support

* Handles `GET` requests
* Returns simple text responses


---
